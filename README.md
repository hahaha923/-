本实验基于 C 语言实现一套完整的伪随 机数生成与随机性检测系统，以 C 标准库 rand()（线 性同余发生器 LCG）为研究对象，采用 rand()%10 生 成区间 [0, 9] 共计 1000000 组整数序列。自主设计并实 现五种经典随机性检 测算法，从分布均匀性、序列相关性、数据重复规律等 维度完成基础统计测评。同时基于拒绝采样实现无偏均 匀随机数，基于 Box-Muller 变换实现正态分布随机数， 完成拓展算法开发。利用蒙特卡洛法求解圆周率 π，量 化对比原生取模随机数与无偏随机数的计算精度差异。 最后将整数序列拆分低 4 位，生成 1200000 比特二进 制流，采用基于 NIST SP800-22 国际标准的密评工具 箱开展密码学级随机性检测。
